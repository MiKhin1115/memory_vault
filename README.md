# Wedding Guest Upload

This project is a pure frontend React + Vite + TypeScript application. The UI is kept in place, but all backend, API, proxy, Netlify, Render, and upload-service logic has been removed from this repository.

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Notes

- This repo contains no backend code and no external endpoint calls.
- Upload submission is currently a frontend placeholder until backend logic is implemented elsewhere.
- `vite.config.ts` and `package.json` are trimmed to frontend-only settings.
